## Installation

Phalcon: ^4 || ^5

PHP: ^7.2 || ^8.0

Require the project using composer:

`composer require "sinbadxiii/phalcon-auth-jwt:dev-master"`

## How use


### Configuration

Copy file from `config/jwt.php` in your folder config and merge your config
